# Screenshots

Pour ajouter des captures d'écran réelles :

1. Ouvrir l'application sur mobile (ou Chrome DevTools en mode device)
2. Prendre une capture de chaque page :
   - `estimer.png` — Page Estimer avec formulaire et résultat
   - `historique.png` — Page Historique avec quelques estimations
   - `comparer.png` — Page Comparer avec deux véhicules comparés
3. Redimensionner à 390px de large (format portrait)
4. Placer les fichiers dans ce dossier
