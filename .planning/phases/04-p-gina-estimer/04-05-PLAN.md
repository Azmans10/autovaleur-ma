<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Marketplaces.getLinks() → anchor.href | URLs generadas por código interno se asignan a atributos href de anchors |
| anchor → external site | El usuario navega a un dominio externo en nueva pestaña |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-05-01 | Tampering | `anchor.href = links.avito` | accept | Las URLs las genera Marketplaces.getLinks() con strings de CAR_DB (datos controlados del código) — ningún input de usuario se inyecta directamente en la URL sin codificación; el slug normaliza con encodeURIComponent internamente |
| T-04-05-02 | Information Disclosure | `target="_blank"` sin rel | mitigate | Todos los anchors incluyen `rel="noopener"` — evita que la pestaña nueva acceda a `window.opener` (tabnapping) |
| T-04-05-03 | Elevation of Privilege | Open redirect | accept | Los dominios de destino están hardcodeados en Marketplaces (avito.ma, moteur.ma, wandaloo.com) — no hay parametrización de dominio por input del usuario |
</threat_model>
