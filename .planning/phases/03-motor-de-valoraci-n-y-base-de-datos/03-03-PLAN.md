# Plan 03-03: Motor Engine.estimate() con formula multiplicativa y breakdown

## Goal

Implementar `var Engine = { estimate: function(params) {...} }` — el punto de entrada unico para valorar un coche. Toma 8 parametros, consulta CAR_DB y DEPRECIATION, aplica la formula multiplicativa (D-11) y retorna el precio estimado con rango y breakdown de 7 lineas.

  <instructions>
Insertar `var Engine = {...};` DESPUES de `var DEPRECIATION` y ANTES del comentario `// ROUTER`.

Anadir comentario separador:

```
    // ===============================================================
    // ENGINE — Motor de valoracion de coches
    // Formula: base_price x f_age x f_km x f_condition x f_fuel
    //          x f_city x f_transmission (D-11)
    // Retorna: { estimated_price, price_range, breakdown } (D-12)
    // ===============================================================
```

Implementar siguiendo EXACTAMENTE el patron del RESEARCH.md (lineas 252-298), con estas precisiones:

1. **Validacion de entrada:** Si `CAR_DB[params.make]` no existe o `CAR_DB[params.make][params.model]` no existe, retornar `null`.

2. **Calculo de edad:** `var currentYear = new Date().getFullYear();` y `var age = Math.max(0, currentYear - parseInt(params.year, 10));`. Clave: `parseInt(params.year, 10)` porque puede venir como string de un `<select>`.

3. **Factor edad (f_age):** `var ageKey = Math.min(age, 15);` y `var f_age = DEPRECIATION.age_factors[ageKey];` — per ALG-02. El floor de 0.30 esta implicitamente cubierto porque age_factors[14] y [15] son 0.30 (per ALG-06).

4. **Factor km (f_km):** Per ALG-03 y RESEARCH:
   ```javascript
   var expectedKm = age * DEPRECIATION.km_per_year;
   var kmDiff = expectedKm - params.mileage;
   var f_km = 1 + (kmDiff / 100000) * DEPRECIATION.km_coefficient;
   f_km = Math.max(DEPRECIATION.km_min, Math.min(DEPRECIATION.km_max, f_km));
   ```
   Usar las constantes de DEPRECIATION en lugar de hardcodear 15000, 0.20, 0.70, 1.30.

5. **Factor condition (f_condition):** `DEPRECIATION.condition_factors[params.condition] || 1.00`

6. **Factor fuel (f_fuel):** `DEPRECIATION.fuel_factors[params.fuel] || 1.00`

7. **Factor city (f_city):** `DEPRECIATION.city_factors[params.city] || 1.00`

8. **Factor transmission (f_transmission):** `DEPRECIATION.transmission_factors[params.transmission] || 1.00`

9. **Multiplicacion en cadena (D-11):**
   ```javascript
   var raw = carData.base_price * f_age * f_km * f_condition * f_fuel * f_city * f_transmission;
   ```

10. **Floor minimo 8000 DH (ALG-06, D-14) + redondeo a centenas:**
    ```javascript
    var estimated_price = Math.max(DEPRECIATION.price_floor, Math.round(raw / 100) * 100);
    ```

11. **Rango +/-10% (ALG-07, D-13):**
    ```javascript
    var price_range = {
      min: Math.round(estimated_price * 0.90 / 100) * 100,
      max: Math.round(estimated_price * 1.10 / 100) * 100
    };
    ```

12. **Breakdown de 7 lineas (D-12) — amount_dh es acumulativo:**
    ```javascript
    var breakdown = [
      { label: 'Prix de base',                    factor: 1.00,           amount_dh: carData.base_price },
      { label: 'Age (' + age + ' ans)',            factor: f_age,          amount_dh: Math.round(carData.base_price * f_age) },
      { label: 'Kilometrage',                      factor: f_km,           amount_dh: Math.round(carData.base_price * f_age * f_km) },
      { label: 'Etat',                             factor: f_condition,    amount_dh: Math.round(carData.base_price * f_age * f_km * f_condition) },
      { label: 'Carburant',                        factor: f_fuel,         amount_dh: Math.round(carData.base_price * f_age * f_km * f_condition * f_fuel) },
      { label: 'Ville',                            factor: f_city,         amount_dh: Math.round(carData.base_price * f_age * f_km * f_condition * f_fuel * f_city) },
      { label: 'Transmission',                     factor: f_transmission, amount_dh: estimated_price }
    ];
    ```
    La ultima linea usa `estimated_price` (con floor aplicado), las demas son calculos acumulativos intermedios.

13. **Retorno (D-12):**
    ```javascript
    return { estimated_price: estimated_price, price_range: price_range, breakdown: breakdown };
    ```

**IMPORTANTE:**
- Usar `var Engine = { estimate: function(params) {...} };` — NO class, NO const.
- Asegurar `parseInt(params.year, 10)` para la conversion de ano.
- Usar constantes de DEPRECIATION (km_per_year, km_coefficient, km_min, km_max, price_floor) en lugar de numeros magicos.

## Verification

- `Engine.estimate()` retorna objeto completo para caso Logan 2019
- `estimated_price` entre 48000 y 70000 DH (caso critico del ROADMAP)
- `breakdown.length === 7`
- Marca/modelo invalido retorna `null`
- Floor funciona: coche muy antiguo (2005) con estado mauvais no baja de 8000 DH
